About Me

I’m a game developer focused on building stylish, performant games using Unity. I enjoy experimenting with mechanics, shaders, and minimal visual design. I believe clean code and strong game feel matter more than noise.

Unity · C# · Game Feel · Pixel Art · UI/UX · Mobile Optimization · Prototyping

Projects

Cannon & Ghosts

My first shipped project on GameJolt. Control a cannon on a movable platform and shoot ghosts in buildings, combining physics-based movement with arcade-style action. Built solo in Unity for Standalone PC.

MagicaVoxel Dino Model

Personal experiment creating a stylized dino model for my upcoming dino runner game. Focused on voxel modeling, color, and form exploration. Not yet published—purely a creative exercise.

Survive

College survival simulation project. Features include flowing grass, a roaming tiger (NavMesh agent), collectible resources updating a UI inventory, chop-able trees, realistic water and sunset shaders, and a custom Unity terrain. First-person immersive experience combining gameplay and visuals.
